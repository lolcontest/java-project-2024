Guide d'utilisation - Game Manager

Raccourcis clavier

Raccourci Action
Ctrl+S Sauvegarder
Ctrl+N Nouvel item
Ctrl+F Rechercher
Delete Supprimer l'item sélectionné
Ctrl+E Exporter
Ctrl+I Importer
Ctrl+T Changer de thème
F1 Afficher l'aide

Format des statistiques

Les statistiques sont au format : statId#minValue#maxValue

Exemple : 7d#3E8#3E8 représente :

  • statId: 7d (dommages)
  • minValue: 3E8 (1000 en hexadécimal)
  • maxValue: 3E8 (1000 en hexadécimal)

Gestion des items

Pour créer un nouvel item :

  1. Cliquez sur le bouton "Nouveau" ou utilisez Ctrl+N
  2. Remplissez les champs requis (ID, Nom, etc.)
  3. Ajoutez des statistiques si nécessaire
  4. Sauvegardez avec Ctrl+S

Export/Import

L'application supporte deux formats d'export :

  • ZIP : Sauvegarde complète des données
  • CSV : Format tableur pour analyse externe
